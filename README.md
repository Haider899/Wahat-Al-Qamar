# WAHAT AL QAMAR

Professional business website for **WAHAT AL QAMAR** — trailer base and chassis manufacturing in the UAE.

## Features

- Modern hero with GSAP text reveal & parallax
- Lenis smooth scrolling
- Product catalog (Flatbed, Lowbed, A-Frame, Skeleton, Curtain Side, Box)
- Projects gallery with category filters
- Contact inquiry form
- Fully responsive (mobile, tablet, desktop)
- UAE branding with Arabic footer tagline

## Run locally

**Option A — Open directly:** Double-click `index.html` in your browser.

**Option B — Local server (recommended for smooth scroll):** Use VS Code Live Server, or:

```bash
npx serve .
```

**Option C — Vite (if Node.js installed):**

```bash
npm install
npm run dev
```

## Deploy

Upload all files to your web host. See `DEPLOYMENT.md` for Apache/IIS/cPanel instructions.

## Customize

- Update phone, email, and address in `index.html` (Contact & Footer sections)
- Replace Unsplash image URLs with your own factory/trailer photos
- Adjust brand colors in `css/main.css` (`:root` variables)
